Narration/description of project:

In this project, i try to replicate the logo of the News website 'Wired'. It is made solely using css and the method is explained below.

Method:

  • First, i create a div container for the texts
  • I then created h1 elements for each letter of the word/logo.
  • Afterwards, i created two classes to be alternated when attributing class names to the h1 elements. This is to make the inverted background-color and color effect possible.
  • I proceed to style the page using internal styling in the header section.

Pitfalls:

  • I did not get the exact font used for the 'wired' logo, i used an alternative, a similar looking font.
  • I could not get the square effect the background colors of each letters had accurately. Note:I will work on that.
  • I think there should/could be an easier, sleekier and more effective way of going about the design. Perhaps through the use of the nth-child pseudoclass. Remember, code should be made as simple as possible.

    Update:The nth-child pseudoclass has now replaced the initial code, making the code simpler, shorter and more elegant.
